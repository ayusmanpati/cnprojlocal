About This Project

This is a project for Computer Networks Laboratory under Dr. Krutika Verma. We are a team of 3 - Ayusman Pati (23051824), Aditya Vikram (23051810) and Abhijeet Patra (23051805). The project is based Operating System's classic Reader-Writer problem, implemented through socket programmming using JavaScript, HTML, CSS and Tailwind. This project aims to understand networks and implement them in our system. It is a prototype of our upcoming project. (P.S. There is an Easter Egg. :P)
